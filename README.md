# Adarsh H Portfolio

Personal portfolio website built with Next.js and deployed on Vercel.

## Live Site

- https://portfolio-coral-seven-57.vercel.app

## About

This portfolio showcases:

- profile and contact information
- data science and machine learning skills
- internship and survey experience
- featured project work
- resume download

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Production Build

```bash
npm run build
npm run start
```

## Deployment

This project is configured to deploy on Vercel through the connected GitHub repository.
